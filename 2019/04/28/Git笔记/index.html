git使用手册

起源

  • 集中式
    1.有集中的版本管理服务器
    2.具备文件版本管理和分支管理能力
    3.继承效率明显地提高
    4.客户端必须时刻和服务器相连
  • 分布式
    1.客户端和服务端都有完整的版本库
    2.脱离服务段，客户端照样可以管理版本
    3.查看历史和版本比较等较多的操作，都不需要访问服务器，比集中式vcs更能提高版本管理效率

名词解释

工作区：在电脑里面能够看到的目录,就是工作区
版本库：工作区里面有一个隐藏目录.git,是Git的版本库
