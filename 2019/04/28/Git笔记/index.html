基本语法

1.创建一个版本库，当前目录下会生成一个.git目录。只能跟踪文本文件的改动，比如txt文件，网页，所有的程序代码等，还可以告诉你每一次的改动。

2.以创建一个hello.java文件为例，将其添加入仓库

3.将文件提交到仓库

4.查看仓库的当前状态

5.查看历史记录,由最近到最远显示

6.版本回退，HEAD表示当前版本,HEAD^表示上一个版本,HEAD^^上上一个版本，也可以输入版本号前4位去回退到任意制定的版本.git回退版本的速度非常快，因为在内部有一个指向当前版本的HEAD指针，回退版本的时候仅仅把HEAD指向目的位置

如果想恢复到以前的版本号，但是忘记了，可以通过git reflog来查看所有的历史记录
